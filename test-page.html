Test 1: Basic Digital Identity Request

Request a verifiable credential using the Digital Credentials API.

const credential = await navigator.credentials.get({ identity: { providers: [{ protocol: "openid4vp", request: { // Credential request parameters } }] } });

Test 2: Request with Specific Claims

Request specific claims from a digital identity wallet.

const credential = await navigator.credentials.get({ identity: { providers: [{ protocol: "openid4vp", request: { requested_claims: { name: true, email: true, age_over_18: true } } }] }, mediation: "required" });

Test 3: Non-Digital Identity Request

Regular credential request (password, etc.) - should NOT be intercepted.

const credential = await navigator.credentials.get({ password: true, mediation: "silent" });
